# Roadmap

## V1

- Application locale complete.
- Donnees fictives prechargees.
- localStorage, export JSON, import JSON et reinitialisation.
- Tableau de bord, etudes, etats, transitions, carte, emotions, catalyseurs, reconnaissances, chronologie, analyse et mode discernement.

## V2

- Remplacer le stockage local par une base de donnees.
- Ajouter une authentification si des donnees personnelles sont stockees.
- Ajouter un historique detaille des modifications.
- Ameliorer les comparaisons d'etats avec une analyse textuelle plus fine.
- Ajouter des exports PDF et CSV.

## V3

- Collaboration multi-utilisateur avec consentement explicite.
- Protocoles d'audit et d'anonymisation.
- Connecteurs optionnels vers des corpus documentaires controles.
