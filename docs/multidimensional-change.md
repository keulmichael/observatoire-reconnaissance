# Changements multidimensionnels

`MultidimensionalChangeEngine` compare les observations chronologiques d'une meme etude sur plusieurs dimensions : Emotion, Attitude, Representation, Behaviour, Relation, Concept, LanguageMarker, Manifestation, Decision, Transmission et Metadata.

Le cas `mepris -> idolatrie` produit une proposition non validee automatiquement :

- attitude anterieure : mepris, devalorisation, polarite negative ;
- etat actuel : idealisation/glorification, polarite positive extreme ou sacralisante ;
- changements : inversion de polarite, representation, relation, vocabulaire, amplification.

Limites conservees : portee du groupe non precisee, contexte mediatique possible, formulation idolatrie potentiellement hyperbolique, besoin de donnees complementaires.
