# Theorie fonctionnelle

L'Observatoire de la Reconnaissance documente des transitions d'etat de comprehension :

```text
S0 -> S1 -> S2 -> ... -> Sn
```

Le coeur de l'application est Delta, c'est-a-dire le chemin observable de la reconnaissance. Une transition Delta relie un etat initial a un etat d'arrivee en conservant les traces suivantes :

- manifestations observees ;
- relations identifiees ;
- emotions associees ;
- catalyseurs ou ponts ;
- formulation de la reconnaissance ;
- transformation concrete ;
- transmission eventuelle ;
- temporalite de maturation ;
- niveau de confirmation.

L'application ne conclut pas qu'une interpretation est vraie. Elle permet de documenter comment une comprehension evolue dans le temps.
