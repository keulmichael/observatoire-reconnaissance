# Guide utilisateur

1. Connectez-vous ou creez un compte.
2. Si des donnees locales existent, utilisez `Migrer mes donnees locales vers mon compte`.
3. Verifiez l'export local telecharge avant migration.
4. Ajoutez ou validez une observation.
5. Ouvrez `Attitudes et representations` ou `Changements multidimensionnels` pour consulter les transformations proposees.
6. Utilisez `Reanalyser toute l'etude` apres l'ajout de nouvelles observations.

Aucune transition ou reconnaissance n'est validee automatiquement.
