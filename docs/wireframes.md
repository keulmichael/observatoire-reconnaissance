# Wireframes

## Structure globale

```text
Sidebar navigation | Vue centrale active | Mode discernement permanent
```

## Vues

- Tableau de bord : grille de statistiques, frequences, dernieres observations, etudes recentes.
- Etudes : liste a gauche, fiche editable a droite.
- Etats : cartes d'etats et comparaison avant / apres.
- Transitions Delta : fiches de transition et sequence en huit phases.
- Carte reflexive : canevas React Flow, filtres, edition, export, legende.
- Emotions : courbe temporelle et repartition.
- Catalyseurs : bibliotheque classee avec frequence et impact.
- Reconnaissances : objets complets avec impacts et confirmation.
- Chronologie : evenements filtres par chemin Delta.
- Analyse : hypotheses locales, metriques et relations recurrentes.

## Responsive

Sur mobile, les trois colonnes deviennent une pile verticale : navigation, contenu, discernement. Les cartes passent en grille simple.
